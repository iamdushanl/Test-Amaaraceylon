# Amaara Cafe and Restaurant

Single-page restaurant website with a cinematic landing experience and modern reservation-focused layout.

## Highlights

- Three-frame hero section with smooth crossfade transitions.
- Clickable and keyboard-accessible hero frame selectors.
- Scroll-aware navigation, reveal animations, and parallax motion.
- Menu, story, reservation form, and footer social links.
